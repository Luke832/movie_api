Documentation Page

Welcome to myFlix. Here you'll be able to find information on movies, directors and actors. You'll also be able to sign up and create lists of your favorite movies!

API Endpoints

Request / Business Logic Endpoint URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET None A JSON object holding data about all movies
Get data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie by title, including: description, genre, director and image URL
Example:
Title: "The Godfather",
Description: "Follows the Corleone crime family in 1940s and 50s New York. It chronicles the transfer of power from patriarch Vito Corleone to his son Michael Corleone",
Genre: {
Title: "Crime",
Description: "Crime films involve various aspects of crime and it's detection. Crime films often overlap with other genres as well, including but not limited to: drama, comedy, mystery, suspense, etc."
},
Director: {

Bio: "Francis Ford Coppola is an American film director, producer and screenwriter. He is widely considered one of the best filmmakers of all time for his work on: The Godfather (1972), The Godfather Part II (1974), The Conversation (1974), and Apocalypse Now (1979), among others.",
Birth: new Date("1939-04-07"),
Death: new Date("")
},
Actors: "Marlon Brando",
ImagePath: "https://m.media-amazon.com/images/M/MV5BM2MyNjYxNmUtYTAwNi00MTYxLWJmNWYtYzZlODY3ZTk3OTFlXkEyXkFqcGdeQXVyNzkwMjQ5NzM@._V1_UY268_CR3,0,182,268_AL_.jpg",
Featured: true,
Get data about a genre by name/title /movies/:genre GET None A JSON object containing the genre description
Example:
Genre: {
Title: "Crime",
Description: "Crime films involve various aspects of crime and it's detection. Crime films often overlap with other genres as well, including but not limited to: drama, comedy, mystery, suspense, etc.",
},
Get data about a director by name /movies/:director GET None A JSON object containing data on the director, including: bio, birth year, and death year (if applicable)
Example:
Director: {
Name: "Francis Ford Coppola",
Bio: "Francis Ford Coppola is an American film director, producer and screenwriter. He is widely considered one of the best filmmakers of all time for his work on: The Godfather (1972), The Godfather Part II (1974), The Conversation (1974), and Apocalypse Now (1979), among others.",
Birth: new Date("1939-04-07"),
Death: new Date("")
},
Allow new users to register /users POST A JSON object holding data about the user to add, including: username, password, email, and date of birth
Example:
Username: "Luke832",
Password: "HelloWorld123",
Email: "Luke832@email.com",
Birthday: new Date("1984-05-26"),
FavoriteMovies: [],
A JSON object holding data about the user that was added
Example:
ID: "123",
Username: "Luke832",
Password: "HelloWorld123",
Email: "Luke832@email.com",
Birthday: new Date("1984-05-26"),
FavoriteMovies: [],
Allow users to update their user info (username) /users/:username PUT A JSON object holding data about the user info to update (username)
Example:
Username: "Luke832456",
Password: "HelloWorld123",
Email: "Luke832@email.com",
Birthday: new Date("1984-05-26"),
FavoriteMovies: [],
A JSON object holding data about the user's new username
Example:
ID: "123",
Username: "Luke832456",
Password: "HelloWorld123",
Email: "Luke832@email.com",
Birthday: new Date("1984-05-26"),
FavoriteMovies: [],
Allow users to add a movie to their favorites list /users/:username/:FavoriteMovies/:Movie_ID POST None
Example:
/users/:Luke832456/:Favorite_Movies/:TheGodfather
A JSON object showing the new movie added to the user's favorites lists
Username: "Luke832456",
Password: "HelloWorld123",
Email: "Luke832@email.com",
Birthday: new Date("1984-05-26"),
FavoriteMovies: ["The Godfather"],
Allow users to remove a movie from their favorites list /users/:username/:FavoriteMovies/:Movie_ID DELETE None
Example:
/users/:Luke832456/:Favorite_Movies/:ForrestGump
A text message saying the movie has been removed from the user's favorites list
Allow users to deregister /users/:username DELETE None
Example:
/users/:Luke832456
A text message saying the user has been deregistered
